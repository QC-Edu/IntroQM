Using Jupyter Notebooks¶

There are several ways to edit a Jupyter notebook (an .ipynb file).

  1. Using Google Colab. Recommended!

  2. Using Binder.

  3. You can, in general, use any online Jupyter Hub (e.g. syzygy). However, it can be difficult to ensure that the dependencies you need are installed. There are lots of other ways to run Jupyter in the cloud.

  4. You can edit Jupyter Notebooks locally. It is relatively easy to use jupyter through Anaconda. One can also use Visual Studio Code to edit notebooks.

We will ensure that assignments work with Google colab. We’ll try to help people using other platforms too, but make no promises.
